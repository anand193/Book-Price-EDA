# Book-Price-EDA
Exploratory data analysis on Book Price Prediction
This project explores the Book Price Prediction dataset.  
I analyzed features like ratings, reviews, book edition, and more.

## Key Insights:
- ratings significantly affects books price
- format and category of books also have strong relation with book price

## Tools Used:
- Python (Pandas, Seaborn, Matplotlib)
- Jupyter Notebook
